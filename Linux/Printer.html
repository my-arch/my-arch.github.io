Add Printer

  1. Install cups, system-config-printer
    • sudo pacman -S cups system-config-printer
  2. Enable and start cups.socket
    • sudo systemctl enable cups.socket
    • sudo systemctl start cups.socket
  3. Install driver
    • Zone 1 UTAR: openprinting-ppds-postscript-ricoh (aur)

External Monitor Setting

Required Package

  1. xorg-xrandr
  2. alsa-utils

Configuration

  1. To read connected screen's properties
    • xrandr
  2. To off laptop's screen and on external monitor
    • xrandr --output eDP1 --off --output HDMI1 --auto
    • xrandr --output eDP1 --off --output HDMI1 --mode 1920x1080

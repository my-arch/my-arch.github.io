Sway Window Manager

Script to Start SwayWM

  • Create a 'startw' executable in scripts folder
  • Create symlink of the file to ~/.local/bin

startw

```console

#!/bin/bash

export GTK_IM_MODULE=fcitx5 export QT_IM_MODULE=fcitx5 export XMODIFIERS=@im=fcitx5 export XDG_CURRENT_DESKTOP=sway export XDG_SESSION_TYPE=wayland export QT_QPA_PLATFORM=wayland export LIBSEAT_BACKEND=logind export _JAVA_AWT_WM_NONREPARENTING=1 export SDL_VIDEODRIVER=wayland export XDG_SESSION_DESKTOP=sway

sway ```

Installation

  • Install libpipewire02, pipewire, pipewire-pulse
  • Install xdg-desktop-portal-wlr
  • Install wireplumber

Chromium Setup

  • Enable WebRTC in chromium
  • Set the env to be XDG_CURRENT_DESKTOP=sway and XDG_SESSION_TYPE=wayland

System Setup

  • systemctl --user enable pipewire.service
  • systemctl --user enable pipewire-media-session.service
  • Uncomment metadata #export metadata api (https://gitlab.freedesktop.org/pipewire/pipewire/-/issues/768#note_812588)

Before Launch Chromium

  • Run:
    • /usr/lib/xdg-desktop-portal -r & /usr/lib/xdg-desktop-portal-wlr &
  • Start chromium

Bulk add users into Microsoft Teams using Powershell under Arch Linux

Installation

  1. yay -S powershell-bin
  2. Install-Module -Name MicrosoftTeams

Add Users to Main Group

  1. Connect-MicrosoftTeams
  2. Get-Team -User chewcy@utar.edu.my
  3. $path='path_to_csv_file'
  4. $GroupId='id_copied_from_get-team'
  5. Import-Csv \(path | % { Add-TeamUser -User \)_.email -GroupId $GroupId }

Add Users to Private Channel

  1. Import-Csv -Path \(path | foreach{Add-TeamChannelUser -GroupId \)GroupdId -DisplayName "YOUR_PRIVATE_CHANNEL_NAME" -user $_.email}

For Windows

  • If Connect-MicrosoftTeams not working, try to relax execution policy:
    • Set-ExecutionPolicy -ExecutionPolicy RemoteSigned
  • To update powershell:
    • install-Module PowerShellGet -Repository PSGallery -Force
  • To install preview version of Microsoft Teams:
    • Install-Module MicrosoftTeams -AllowPrerelease -RequiredVersion "1.1.9-preview"

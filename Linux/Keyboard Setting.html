Keyboard Setting

Keyboard Layout

  1. To set keyboard layout to
    • Primary - US keyboard
    • Variant - Dvorak
    • Command - setxkbmap -layout us,us -variant ,dvorak -option grp:alt_shift_toggle

Keyboard Key Setting

  1. To swap key
    • Install xmodmap
    • Create xmodmap config file - touch ~/.Xmodmap
    • To reload, run: xmodmap ~/.Xmodmap
  2. To read keycode
    • Install xbindkeys
    • To read keycode, run: xbindkeys -k
    • To reload, run: xbindkeys -p

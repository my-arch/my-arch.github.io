Screenshot and Screencast

Screenshot

  • Scrot is needed for screenshot
    1. Whole screen
      • scrot filename.png
    2. Snipping
      • scrot filename.png -s

Screencast

  • ffmpeg is needed for screencast
    1. Loss-less video
      • ffmpeg -f x11grab -video_size 1366x768 -i $DISPLAY -f alsa -i default -c:v ffvhuff -c:a flac output.mkv
    2. Resize
      • ffmpeg -f x11grab -s 1366x768 -r 25 -i $DISPLAY -f alsa -i default test.mp4
    3. Split video (with "copy" is to disable re-encoding")
      • ffmpeg -i input.mp4 -vcodec copy -acodec copy -ss 00:01:00.000 -t 00:00:10.000 output.mp4

Webcam Recording

  • ffmpeg -f v4l2 -framerate 25 -video_size 1280x720 -i /dev/video0 filename.mkv

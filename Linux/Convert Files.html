Convert Files

Manipulation of PDF Files

  1. Change quality of pdf file
    • gs -dNOPAUSE -dBATCH -dFirstPage=FirstPage -dLastPage=LastPage -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 -dPDFSETTINGS=setting -sOutputFile=output.pdf input.pdf
    • change "setting" to
      • /screen
      • /ebook
      • /printer
      • /prepress
  1. Merge/split pdf files
    • convert input1.pdf input2.pdf output.pdf
    • convert input.pdf[0-2] output.pdf
  1. Merge using ghostscript
    • gs -dNOPAUSE -sDEVICE=pdfwrite -sOUTPUTFILE=combine.pdf -dBATCH 1.pdf 2.pdf
  1. Flatten PDF
    • gs -sDEVICE=pdfwrite -dPDFSETTINGS=/default -dNOPAUSE -dQUIET -dBATCH -sOutputFile=flattened.pdf input.pdf
  1. Flatten PDF with no-copy
    • gs -o output.pdf -dNoOutputFonts -sDEVICE=pdfwrite input.pdf

Convert DJVU to PDF

  1. DJVU manipulation
    • ddjvu --format=pdf inputfile.djvu ouputfile.pdf

Convert PDF to JPEG

  1. Convert file using libreoffice
    • libreoffice --headless --invisible --convert-to jpeg *.pdf

Zip File/Folder

  1. Zip folder
    • zip -er Folder.zip Folder
  1. Zip folder with password
    • zip -P Password -r Folder.zip Folder

Change File Permission

  1. find [yourdir] -type f -exec chmod 644 {} \;
  2. find [yourdir] -type d -exec chmod 755 () \;

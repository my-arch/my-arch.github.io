TM Router Configuration

  1. RG IP address: 192.168.0.11.User Name: admin & Password : ‘(blank)’
  2. Click on Login to Proceed
  3. Click on the Wizard
  4. Select to begin step-by-step configuration
  5. Key in PPPoEUser Name
  6. Key in the PPPoEPassword
  7. Confirm the PPPoEPassword
  8. Click on the Nextbutton
  9. Enable Wireless
  10. Edit default SSID (Using PPPoEUser Name) to your preferred SSID
  11. Click on the Nextbutton
  12. Enable wireless security. If you select Disable, click Nextto proceed to the next page.
  13. Select Wireless Authentication –“WPA-PSK/WPA2-PSK”are much securer, since WPA has some improvements over WEP.
  14. Select the key type
  15. Set the Pre-shared key.
  16. Click on the Nextbutton
  17. Verify that the set information is correct and then click Submitto save the settings.To modify the settings in the previous pages, click Back.
